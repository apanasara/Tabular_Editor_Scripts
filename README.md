## 🚀 Getting Started

1. 📥 **Clone or Download** this repo  
2. 🏗️ **Open Tabular Editor > Advanced Scripting**  
3. 📝 **Copy in your chosen script from `/C# Scripts/`**  
4. 🎯 **Run and automate your Power BI project!**
